sync_alt Data Binding

Mecanismo para conectar datos entre el componente TypeScript y el template HTML, en una o ambas direcciones.

call_split Comunicación entre Componentes

Técnicas para intercambiar datos entre componentes padre e hijo utilizando decoradores y servicios.

alt_route Routing

Sistema de navegación basado en URLs para mostrar distintos componentes según la ruta activa.

extension Directivas

Instrucciones especiales en el template que permiten modificar la estructura o el comportamiento de elementos HTML.
