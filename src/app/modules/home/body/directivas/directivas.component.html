extension Directivas en Angular - Conceptos Básicos

help_outline ¿Qué son las Directivas?

Las directivas son instrucciones para modificar el DOM, cambiar estilos, comportamientos o estructura de elementos.

Angular tiene tres tipos principales: estructurales, atributivas y componentes.

Ver documentación oficial

view_week Directivas Estructurales

Cambian la estructura del DOM agregando o removiendo elementos.

Ejemplos comunes: *ngIf, *ngFor. 

      <!-- Mostrar solo si está activo -->
      <div *ngIf="activo">¡Activo!</div>

      <!-- Listar elementos -->
      <ul>
        <li *ngFor="let item of items">{{ item }}</li>
      </ul>

format_paint Directivas Atributivas

Cambian la apariencia o comportamiento de un elemento existente sin modificar su estructura.

Ejemplos comunes: [ngClass], [ngStyle]. 

    <!-- Cambiar clases CSS dinámicamente -->
    <div [ngClass]="{'resaltado': esResaltado}">Texto con clase dinámica</div>

    <!-- Cambiar estilos dinámicamente -->
    <p [ngStyle]="{'color': colorTexto, 'font-weight': 'bold'}">Texto estilizado</p>

build Directivas Personalizadas

Puedes crear tus propias directivas para encapsular comportamientos reutilizables. 



import { Directive, ElementRef, Renderer2, HostListener } from '@angular/core';

@Directive({
  selector: '[appResaltar]'
})
export class ResaltarDirective {
  constructor(private el: ElementRef, private renderer: Renderer2) {}

  @HostListener('mouseenter') onMouseEnter() {
    this.renderer.setStyle(this.el.nativeElement, 'backgroundColor', 'yellow');
  }

  @HostListener('mouseleave') onMouseLeave() {
    this.renderer.removeStyle(this.el.nativeElement, 'backgroundColor');
  }
}

Luego úsala así en HTML:

¡Pasa el mouse para resaltar!
